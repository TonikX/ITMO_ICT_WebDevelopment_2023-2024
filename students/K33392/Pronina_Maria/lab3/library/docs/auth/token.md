# Создание токена для пользователя

Позволяет создать токен авторизации для пользователя

***URL*** : `/api/auth/token`

***Method*** : `POST`

***Auth required*** : YES

***Permission required*** : None

## Success Responses

**Code** : `200 OK`

**Content** :

```json
{
    "token": "60478bf9a8be3e6342d0592fba6aa94f0e946f74"
}
```

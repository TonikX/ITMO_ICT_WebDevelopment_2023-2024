# Регистрация нового пользователя

Позволяет зарегистрировать нового пользователя

***URL*** : `/api/auth/users`

***Method*** : `POST`

***Auth required*** : YES

***Permission required*** : None

## Success Responses

**Code** : `201 Created`

**Content** :

```json
{
    "email": "",
    "username": "roma",
    "id": 5
}
```

# Добавление новой копии

Позволяет добавить новую копию

***URL*** : `/copys/new`

***Method*** : `POST`

***Auth required*** : YES

***Permission required*** : None

## Success Responses

**Code** : `201 Created`

**Content** :

```json
{
    "id": 6,
    "book": 3,
    "hall": 2
}
```

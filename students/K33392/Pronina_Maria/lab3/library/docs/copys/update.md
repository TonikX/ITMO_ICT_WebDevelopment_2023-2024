# Изменение информации о копии

Позволяет обновить информацию о копии

***URL*** : `copys/update/<int:pk>`

***Method*** : `PUT`

***Auth required*** : YES

***Permission required*** : None

## Success Responses

**Code** : `200 OK`

**Content** :

```json
{
    "id": 1,
    "book": 1,
    "hall": 1
}
```

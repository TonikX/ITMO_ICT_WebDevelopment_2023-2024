# Показать информацию обо всех залах

Выводит полную информацию обо всех залах

***URL*** : `api/halls/all`

***Method*** : `GET`

***Auth required*** : YES

***Permission required*** : None

## Success Responses

**Code** : `200 OK`

**Content** :

```json
[
    {
        "id": 1,
        "sequence_number": 1,
        "name": "British Literature",
        "capacity": 30
    }
]
```

# Показать профессию всех войнов

Выводит информацию о профессии всех войнов

***URL*** : `/warriors/skills`

***Method*** : `GET`

***Auth required*** : YES

***Permission required*** : None

## Success Responses

**Code** : `200 OK`

**Content**:

```json
{
        "name": "Олег",
        "profession": {
            "id": 1,
            "title": "Дворник",
            "description": "Дворнит"
        }
    }
```

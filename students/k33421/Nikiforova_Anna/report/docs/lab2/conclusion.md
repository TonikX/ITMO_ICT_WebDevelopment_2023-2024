В данной работе мне удалось овладеть практическими навыками и умениями реализации web-сервисов
средствами Django. 

Был разработан веб-сайт для учета домашних заданий. Был разработан интерфейс для ученика и для преподавателя. 

Функционал включает в себя переход по ссылкам, фильтрацию и поиск, пагинацию, отправку и валидацию форм. 

Также для создания фронт-части веб-сайта был использован Bootstrap.
